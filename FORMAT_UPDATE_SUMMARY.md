## ✨ Teacher Registration Workflow (After Update)

1. **Admin Workflow**:
   - Admin calls `POST /api/admin/registration-codes` with teacher details
   - System generates sequential code: `tch-2026-001`, `tch-2026-002`, etc.
   - Admin sends code to teacher via email or manual distribution

2. **Teacher Workflow**:
   - Teacher visits `/register/teacher`
   - Enters: Email, Full Name, Password, Registration Code (`tch-2026-001`)
   - System validates code format and existence
   - Creates user with:
     - Username: `tch-2026-001` (same as code)
     - Email: Provided by teacher
     - Password: Hashed and stored
     - Allocations: Subjects and form class from code
   - Sends welcome email with username
   - Marks code as USED

3. **Teacher Login**:
   - Username: `tch-2026-001` (the registration code)
   - Password: Set during registration
   - Accesses dashboard with auto-assigned subjects and form class
